# Lumen internal APIs

First APIs will be **internal**, because we prioritize the Important APIs first.

## System internal APIs (Locates in /sysmain/os/internal/api/internal/system)

- **Debug API**: Debug Helper.
- **Sys API**: General system & buses management.
- **UI API**: Manages LumenGUI.
- **Web API**: Access to Web extensions.

## Kernel internal APIs (Locates in /sysmain/os/internal/api/internal/k)

- **Debug API**: Kernel debug helper.
- **HW API**: Hardware management.
- **Interfaces API**: Manages Lumen kernel interfaces.
- **Storage API**: Main Storage Management API.

As for now Lumen M2, these APIs is Currently starting development.
